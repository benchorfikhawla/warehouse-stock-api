# Warehouse Stock API

**Version:** 1.0  
**Author:** Khawla  
**GitHub:** https://github.com/benchorfikhawla/warehouse-stock-api  

## 📦 Overview

The **Warehouse Stock API** is an Odoo 16 module that provides a simple REST API to access warehouse stock information.  
This module is useful for integrating Odoo inventory data with external tools, dashboards, or e-commerce platforms.

## 🚀 Features

- 🔍 Get stock quantities per warehouse
- 📊 Retrieve stock movement details
- 🔌 Ready for integration with external systems
- ⚙️ Supports multi‑warehouse inventory
